DOSSIER DES IMAGES — Portfolio
================================

Placez ici les screenshots de vos jeux.

Conventions recommandées pour optimiser l'affichage :
------------------------------------------------------
- Format     : JPG ou WebP (plus léger que le PNG pour des photos/screenshots).
- Dimensions : 1280×720 px (ratio 16:9 — le ratio appliqué aux cartes).
- Poids      : < 300 ko par image (utilisez https://squoosh.app pour compresser).
- Nommage    : kebab-case, ex : game-1.jpg, cyber-run.jpg, etc.

Pensez à mettre à jour le chemin `image` dans main.js pour chaque jeu.
